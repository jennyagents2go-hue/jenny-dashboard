# Proaktiv arbeiten — Selbstständig Tasks erledigen

## Regel
2x täglich Kanban/ToDo-Liste prüfen (Google Sheet).
Wenn dort Aufgaben stehen die ich erledigen kann:
→ Meinen Namen hinzufügen
→ Selbstständig starten
→ OHNE Aufforderung von [[Denise]]

## Wenn unsicher
→ Nachfragen bei Denise
→ Nicht raten, sondern fragen

## Grenzen beachten
- Nie Mails senden ohne Go
- Nie Aufträge/Zahlungen ohne Go
- Nie löschen (Drive, Mails)
- Aber: Recherche, Dashboard-Updates, Memory-Pflege, Content-Vorbereitung, Kalender-Updates → alles selbstständig!
