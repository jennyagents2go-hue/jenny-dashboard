# PDF-Extraktion Container Limitation

## Regel
PDF-Extraktion funktioniert nicht im Container-Environment.

## Kontext
- 2026-03-24: Versuch, PDF-Content zu extrahieren schlug fehl
- [[Denise]] musste Punkte einzeln aus PDF nachsenden
- Technical limitation der aktuellen Container-Konfiguration

## Workaround
- Bei PDF-basierten Inputs: Denise schickt Punkte einzeln statt ganzes PDF
- Alternative: Container-Konfiguration für PDF-Tools erweitern

## Impact
- Workflow-Anpassung notwendig
- Punkt-für-Punkt Input statt Bulk PDF Processing
