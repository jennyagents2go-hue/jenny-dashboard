# PDF-Extraktion im Container

## Problem
PDF-Extraktion funktioniert nicht im Container-Environment.

## Lösung/Workaround
- [[Denise]] sendet Inhalte punkt-für-punkt einzeln
- Vermeidung von PDF-Upload, stattdessen strukturierte Text-Eingabe

## Status
- Bestätigt 2026-03-24
- Alternative Arbeitsweise etabliert

## Kontext
- Aufgetreten bei TK100X Implementation
- 13 Punkte wurden einzeln übertragen statt PDF-Upload
