# LinkedIn Posting Learnings

## Post 1 — Comeback (24.03.2026)
- Reach Rate 135% — Algorithmus belohnt Comeback nach langer Pause
- Engagement 2,4% — knapp am Benchmark, solide
- **Kommentare niedrig (2)** → IMMER mit einer Frage enden!
- 23 Profilbesuche → Neugier geweckt, Profil muss sitzen
- 0 neue Follower → kommt mit Post 2-3

## Regeln für alle Posts (aus Learnings)
1. **Immer mit Frage oder Call-to-Action enden** → provoziert Kommentare
2. **Echte Fotos > Grafiken** → mehr Reichweite
3. **Kurze Sätze, viel Weißraum** → mobile-optimiert
4. **Erste 2 Zeilen = Hook** → muss zum "Mehr anzeigen" klicken zwingen
5. **Hashtags: 5-8 am Ende** → nicht im Text
6. **Posten Mo/Mi/Do morgens** → beste Zeiten für LinkedIn DACH
7. **Video-Posts = 3x Engagement** → wenn möglich Video nutzen
