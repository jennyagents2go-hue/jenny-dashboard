# ⚡ SKILL: Morning Brief
# Morning Brief — Täglicher Workflow

## Wann
- Täglich 07:00 Wien per Mail (Cron-Job)
- Nochmal im Chat wenn [[Denise]] "Guten Morgen" schreibt

## Format (EXAKT so)

```
🌤 MORNING BRIEF — [Datum]
Wetter in Lieboch: [Temperatur, Bedingungen]

📅 HEUTE:
- Wichtigster Termin: ...
- Kritischer Punkt: ...
- [Weitere wichtige Tasks]

🌍 WELT:
1. ...
2. ...
3. ...

🤖 AI:
1. ...
2. ...
3. ...

🎯 FOKUS HEUTE:
- Wenn du nur eine Sache schaffst → ...

⚠️ ACHTUNG:
- [Was muss Denise heute beachten]

⛔ IGNORIEREN:
- [Was soll sie heute ignorieren]
```

## Regeln
- Maximal kompakt
- Nur relevante Informationen
- Fokus klar priorisieren
- Datum + Wetter in Lieboch immer oben
- Postings NUR am Tag des Postens erwähnen, nicht vorher
- Tasks aus Kanban prüfen (was ist wirklich offen?)
