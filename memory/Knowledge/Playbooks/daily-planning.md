# ⚡ SKILL: Daily Planning

## Wann
- Wenn [[Denise]] "Guten Morgen" schreibt (Wochentag)
- Nach dem Morning Brief (der kommt per Mail um 07:00)

## Format (EXAKT so)

```
🔥 HEUTE

agents4you:
1. ...
2. ...

Einzelunternehmen:
1. ...

⛔ NICHT HEUTE:
- ...

⚡ FOKUS:
- ...

📅 KALENDER:
- ...
```

## Regeln
- Maximal 3 Hauptaufgaben
- Nach agents4you und Einzelunternehmen trennen
- Überlastung prüfen und reduzieren
- Fokus auf wenige, wichtige Tasks
- Keine Überplanung
- Tasks aus Kanban + Working.md + Memory prüfen
- Mitdenken als COO! Was fehlt? Was ist überfällig?
- Proaktiv challengen wenn zu viel auf dem Teller liegt

## Midday Check-In
- Um 12:00 Wien aktiv nachfragen: "Woran hast du bis jetzt gearbeitet?"
- Status updaten, Rest des Tages anpassen wenn nötig

## Datenquellen
- Kanban Board (Default-Cards im Dashboard)
- Working.md
- Memory/Projects/
- Google Calendar
- Content-Plan (Postings nur am Posting-Tag!)
