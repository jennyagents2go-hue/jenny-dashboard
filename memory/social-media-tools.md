# Social Media Content Creation — Kostenlose Tools & Workflow

## Bildgenerierung (kostenlos)
- **Bing Image Creator / Microsoft Copilot** — DALL-E 3, komplett kostenlos, keine Limits → https://www.bing.com/images/create
- **Leonardo.AI** — 150 Credits/Tag, hochwertig → https://leonardo.ai
- **Adobe Express** — Text-zu-Bild + Design-Templates → https://express.adobe.com
- **Canva** — Design-Templates, CI-Vorlagen, kostenlos → https://canva.com
- **Lexica.art** — Stable Diffusion, Prompt-Inspiration → https://lexica.art
- **SeaArt.AI** — Bild + Video aus Bild → https://seaart.ai

## Videogenerierung (kostenlos)
- **CapCut** — Text-zu-Video, Bild-Animation, Schnitt, KOMPLETT kostenlos → https://capcut.com
- **Pika Labs** — Text-zu-Video, Bild-zu-Video, kostenlose Beta → https://pika.art
- **RunwayML Gen-2** — 125 Credits/Monat → https://runwayml.com
- **Leonardo.AI Motion** — Bild zu kurzem Video → https://leonardo.ai
- **SeaArt.AI Video** — Bild zu Video → https://seaart.ai

## Workflow für Denise's LinkedIn-Postings
1. Jenny erstellt Text + Prompt
2. Bild generieren via Bing Image Creator oder Leonardo.AI
3. In Canva CI-Vorlage packen (Branding, Farben, Logo)
4. Für Video: CapCut oder Pika Labs
5. Fertiges Posting (Text + Bild/Video) an Denise liefern
