📱 Jenny iPhone Icon

Icon für den iOS Homescreen - Rechtsklick → "Bild speichern unter"

📥 Download (1024x1024 für iOS)

📱 So fügst du das Icon hinzu:

  1. Icon herunterladen (Rechtsklick → Bild speichern)
  2. Safari öffnen → Dashboard-URL aufrufen
  3. Teilen-Button → "Zum Home-Bildschirm"
  4. Name auf "Jenny" ändern
  5. Gespeichertes Icon auswählen
