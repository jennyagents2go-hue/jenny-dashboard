📲 So gehts (super einfach!):

1. Screenshots App öffnen
2. Screenshot von dem Icon oben machen
3. Fotos App → Screenshot bearbeiten
4. Dein Gesicht in den weißen Kreis einfügen
5. Shortcuts App → "+" → "App öffnen" → Safari
6. Aktion hinzufügen: "URL öffnen" → Dashboard URL
7. Icon ändern → dein bearbeitetes Bild wählen
8. Name: "Jenny 🔥" → Fertig!
